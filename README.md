# Superstore Sales Analysis

## Overview
This project presents an end-to-end sales analysis of the Superstore dataset, focusing on customer segmentation, VIP customer behavior, and order returns.  
The goal of the analysis is to identify high-impact customers and evaluate return patterns to support data-driven business decisions.

## Dataset
- Source: Superstore Sales Dataset (Kaggle)
- Data includes orders, customers, products, sales, and returns
- Granularity: order-level data

## Key Metrics & Analysis
- Order volume and sales value
- ABC customer segmentation based on sales contribution
- VIP vs Non-VIP customer distribution
- Returned orders share and impact

## Methodology
1. Data cleaning and preparation using Google Sheets
2. Customer segmentation with ABC analysis
3. Identification of VIP customers
4. Returns analysis based on negative sales values
5. Data visualization and insight generation

## Key Insights
- VIP customers represent a small share of the customer base while contributing disproportionately to sales
- The share of returned orders is relatively low compared to total transactions
- Customer segmentation enables more focused and efficient sales strategies

## Tools Used
- Google Sheets (data processing and analysis)
- Google Slides (data visualization and presentation)
